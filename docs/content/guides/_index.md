Each guide is built around a job rather than a command: pulling a span of monthly
bulk dumps, acquiring one subreddit or account, querying the imported records,
shaping the output, publishing to Hugging Face, and reading the hardware budget.
They assume you have run the [quick start](/getting-started/quick-start/).
